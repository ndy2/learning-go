---

아래는 fmt 패키지의 `Stringer` 인터페이스를 보여준다.

```go
type Stringer interface {
    String() string
}
```

인터페이스는 이름의 마지막에 `er`를 붙여서 명명되는 것이 관례이다. 
이는 인터페이스가 특정한 동작을 수행하는 타입을 나타낸다는 것을 나타낸다.

e.g.) `Reader`, `Writer`, `Stringer`, `Logger` 등

인터페이스는 메서드 집합을 정의한다.
