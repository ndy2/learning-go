### 10.5.1 동시성 없이 API 를 유지

동시성은 구현 세부 사항 (implementation detail) 이다. API 를 설계할 때 동시성을 외부로 노출하지 말자.

즉, 채널이나, 뮤텍스를 API 의 타입, 함수 메서드에서 드러내지 않도록 하자.
